### **Tarefa 3: Utilitário de conversão de temperatura**

Nesta tarefa, você criará um programa que converte temperaturas entre Celsius e Fahrenheit. O usuário poderá inserir várias temperaturas e escolher a direção da conversão. O programa deve usar loops para processar a entrada e continuar rodando até que o usuário escolha sair.

#### **Para alcançar uma nota D**

- Escreva um programa que converte uma única temperatura de Celsius para Fahrenheit.
  - Solicite ao usuário que insira uma temperatura em Celsius.
  - Converta a temperatura para Fahrenheit usando a fórmula: `F = C * 9/5 + 32`.
  - Exiba a temperatura convertida.
  - Você precisa fornecer capturas de tela do seu código e dos diferentes resultados que ele pode gerar.
  - Utilize títulos apropriados no portfólio para garantir que essa tarefa seja facilmente identificada junto ao restante do seu trabalho.

---

#### **Para alcançar uma nota C**

Complete todos os passos anteriores e, em seguida:
- Melhore o programa adicionando um loop que permita ao usuário converter várias temperaturas até que ele escolha sair.
- Após cada conversão, pergunte ao usuário se ele deseja converter outra temperatura. Se escolher continuar, repita o processo com uma nova entrada. Se escolher sair, termine o programa.
- Mantenha o controle da quantidade de conversões realizadas.
- Exiba a contagem final quando o usuário decidir sair (por exemplo, "Você converteu X temperaturas").
- Você precisa fornecer capturas de tela do código e dos diferentes resultados que ele pode gerar.
- Certifique-se de que o seu documento Word tenha títulos apropriados para identificar essa tarefa junto ao restante do seu trabalho.

#### **Para alcançar uma nota B**

Complete todos os passos anteriores e, em seguida:
- Modifique o programa para lidar tanto com conversões de Celsius para Fahrenheit quanto de Fahrenheit para Celsius.
- Solicite ao usuário que escolha a direção da conversão (Celsius para Fahrenheit ou Fahrenheit para Celsius) antes de inserir a temperatura.
- Use instruções condicionais para realizar a conversão escolhida:
  - Para Celsius para Fahrenheit: `F = C * 9/5 + 32`.
  - Para Fahrenheit para Celsius: `C = (F - 32) * 5/9`.
- Exiba uma mensagem indicando a direção da conversão e a temperatura convertida.
- Forneça dicas baseadas no valor da temperatura:
  - Se estiver convertendo para Fahrenheit, indique se a temperatura resultante está acima ou abaixo do ponto de congelamento (32°F).
  - Se estiver convertendo para Celsius, indique se a temperatura resultante está acima ou abaixo do ponto de congelamento (0°C).
